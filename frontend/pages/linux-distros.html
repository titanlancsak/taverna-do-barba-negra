Linuxディストリビューション

人気のある無料・オープンソースの Linux ディストロ公式ダウンロードリンク。

  • Rocky Linux — エンタープライズ向け、RHEL互換。
  • Ubuntu — 最も人気のあるデスクトップ／サーバー用ディストロ。
  • Debian — 安定志向でコミュニティ主導。
  • Fedora — 最先端、Red Hat がスポンサー。
  • Linux Mint — 初心者に優しいデスクトップ体験。
  • Arch Linux — ミニマルで高いカスタマイズ性。
  • Kali Linux — セキュリティとペネトレーションテスト向け。
