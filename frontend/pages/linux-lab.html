Linuxラボ

ブラウザ上で動く使い捨ての Linux 環境です。コマンドを自由に試せます。ページを閉じるとすべて消えます。

⚠️ この環境はインターネットに接続されていません。 すべてあなたのブラウザ内だけで動くため、ネットワークを必要とするコマンドは動きません：

  • パッケージのインストール（apk add / apt install など）
  • ping、curl / wget でのダウンロード、git clone など

試せること： ファイル操作（ls, cd, mkdir, rm）、テキスト処理（cat, grep, sed, vi）、権限（chmod）、シェルスクリプトなど、ローカルで完結するコマンドは自由に試せます。

すべてブラウザ内で動くのでサーバーには何も残りません。ページを閉じる・再起動するとすべてリセットされます。初回はイメージの読み込みに少し時間がかかります。画面をクリックしてから入力してください。
