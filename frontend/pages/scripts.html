便利な自動化スクリプト

ターミナルにコピー＆ペーストして使ってください。実行する前に必ずスクリプトを読みましょう。

システム更新（Rocky／RHEL系）

sudo dnf update -y && sudo dnf autoremove -y

ディスク上の大きなファイルを探す

du -ah / 2>/dev/null | sort -rh | head -20

開いているポートを確認

sudo ss -tulpn

タイムスタンプ付きでフォルダをバックアップ

tar -czf backup_$(date +%Y%m%d_%H%M%S).tar.gz /path/to/folder

システムリソースをリアルタイム監視

htop

ダウンロードフォルダをファイル種別で自動整理

mkdir -p ~/Downloads/{Images,Videos,Audio,Documents,Archives,Scripts}
cd ~/Downloads
for f in *; do
  [ -f "$f" ] || continue
  case "${f,,}" in
    *.jpg|*.jpeg|*.png|*.gif|*.webp|*.svg) mv "$f" Images/ ;;
    *.mp4|*.mkv|*.mov|*.avi|*.webm) mv "$f" Videos/ ;;
    *.mp3|*.wav|*.flac|*.ogg|*.m4a) mv "$f" Audio/ ;;
    *.pdf|*.docx|*.txt|*.xlsx|*.pptx) mv "$f" Documents/ ;;
    *.zip|*.tar|*.gz|*.rar|*.7z) mv "$f" Archives/ ;;
    *.sh|*.py|*.js) mv "$f" Scripts/ ;;
  esac
done
echo "Downloads folder organized!"

フォルダを監視して新しいファイルを自動仕分け（リアルタイム）

sudo dnf install inotify-tools -y
mkdir -p ~/Downloads/{Images,Videos,Audio,Documents,Archives}
inotifywait -m -e create --format '%f' ~/Downloads | while read FILE; do
  case "${FILE,,}" in
    *.jpg|*.jpeg|*.png|*.gif) mv ~/Downloads/"$FILE" ~/Downloads/Images/ ;;
    *.mp4|*.mkv|*.mov) mv ~/Downloads/"$FILE" ~/Downloads/Videos/ ;;
    *.mp3|*.wav|*.flac) mv ~/Downloads/"$FILE" ~/Downloads/Audio/ ;;
    *.pdf|*.docx|*.txt) mv ~/Downloads/"$FILE" ~/Downloads/Documents/ ;;
    *.zip|*.tar|*.gz) mv ~/Downloads/"$FILE" ~/Downloads/Archives/ ;;
  esac
done

フォルダ内の30日より古いファイルを削除

find /path/to/folder -type f -mtime +30 -delete

ディスク使用量の概要を確認

df -h --total | grep total

実行中のサービスを一覧表示

systemctl list-units --type=service --state=running

スワップファイルを作成（低メモリのVMに便利）

sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

フォルダ内の全画像を圧縮（ImageMagickが必要）

sudo dnf install ImageMagick -y
cd /path/to/folder
mkdir -p compressed
for f in *.jpg *.jpeg *.png; do
  [ -f "$f" ] || continue
  convert "$f" -quality 75 -resize 1920x1920\> "compressed/$f"
done
echo "Images compressed into ./compressed"

特定のポートを使うプロセスを見つけて終了

sudo lsof -i :PORT_NUMBER
# then run:
sudo kill -9 PID

SSL証明書の自動更新（Certbotのcron確認）

sudo certbot renew --dry-run
